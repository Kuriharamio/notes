## 自动加载值：

​	**定时器的自动加载值是指每隔一定的时间间隔，定时器会自动执行一次特定的任务**。

​	`__HAL_TIM_SetAutoreload(&htim2, us);		//设置定时器自动加载值`

## 定时器初始值：

 	**定时器的初始值是指定时器被设置时的初始值，通常用于控制定时器的工作时间或计数器**

​	`uint_16 counter = 0;`

​	`__HAL_TIM_SetCounter(&htim1, counter); 		//设置定时器初始值`

## 获取定时器当前计数：

​	`counter = __HAL_TIM_GetCounter(&htim1); // 获取定时器当前计数`

## 停止计时器：

​	`HAL_TIM_Base_Stop(&htim2); // 停止定时器`

## 使用定时器实现微妙延时：

```
	`void delay_us(uint16_t us)`
	{
		uint16_t counter = 0;
		__HAL_TIM_SetAutoreload(&htimx,us);
		__HAL_TIM_SetCounter(&htimx,counter);
		HAL_TIM_Base_Start(&htimx);  //启动计时器
		
		while(counter != us)
		{
			counter = __HAL_TIM_GetCounter(&htimx);
		}
		HAL_TIM_Base_Stop(&htimx);  //停止计时器
	}
```
