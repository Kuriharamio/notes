  可知 `enp12s0` 为有线连接网口

- ```bash
  ls /etc/netplan/
  ```

  查看文件名，输出

  ```bash
  01-network-manager-all.yaml
  ```

- 修改该文件

  ```bash
  sudo vim /etc/netplan/01-network-manager-all.yaml
  ```

  ```bash
  # Let NetworkManager manage all devices on this system
  network:
    version: 2
    renderer: NetworkManager
    ethernets:
      enp12s0:
        dhcp4: no
        addresses:
          - 192.168.1.2/24
        gateway4: 192.168.1.1
        nameservers:
          addresses:
            - 8.8.8.8
            - 8.8.4.4
  
  ```

- 启用配置

  ```bash
  sudo netplan apply
  ```

  

3、[安装 `Livox ROS Driver 2`](https://github.com/Livox-SDK/livox_ros_driver2/blob/master/README.md)

> 注意：
>
> 在下载完 `Livox ROS Driver 2` 之后，先修改 `ws_livox/src/livox_ros_driver2/config/MID360_config.json`中的 `IP` 配置
>
> `host IP` 为刚刚设置的 192.168.1.2
>
> `lidar IP` 需要启动 `LivoxViewer2 for Ubuntu v2.3.0`（去官网下载），查看日志输出来确定（或者看雷达上的二维码尾号）
