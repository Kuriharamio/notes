# 模电（十）：直流电源

直流电源将交流电转换为稳定的直流电，主要由整流、滤波和稳压电路组成

<img src="模电（十）：直流电源.assets/image-20241227225257150.png" alt="image-20241227225257150" style="zoom:67%;" />

## 一、整流电路：

- 利用二极管的单向导电性，将交流电转换为单一方向的脉动电压


* **类型**（以下均输入峰值为 $\sqrt2U_2$ 的正弦波）

    * **单相半波整流电路**
    
        <img src="模电（十）：直流电源.assets/image-20241227230025320.png" alt="image-20241227230025320" style="zoom:67%;" />
    
        * **输出电压有效值：** $\bar U_o = 0.45U_2\\$  
        * **输出电流有效值：** $\bar I_{o} = \frac{\bar U_o}{R_L}=\frac{0.45U_2}{R_L}\\$ 
        * **二极管电流应力：** $I_{D} = 1.1I_o\\$
        * **二极管反向电压（电压应力）：** $U_{D} = 1.1\sqrt2 U_2$
    
        
    
    * **单相全波整流电路**
      
        <img src="模电（十）：直流电源.assets/image-20241227231606128.png" alt="image-20241227231606128" style="zoom: 67%;" />
        
        * **输出电压有效值：** $\bar U_o = 0.9U_2\\$  
          * 为单相半波的2倍
        * **输出电流有效值：** $\bar I_{o} = \frac{\bar U_o}{R_L}=\frac{0.9U_2}{R_L}\\$ 
          * 为单相半波的2倍
        * **二极管电流应力：** $I_{D} = 1.1\frac{I_o}2\\$
          * 只有半个周期导通，但是平均电流又翻倍，所以电流应力和单相半波整流相同
        * **二极管反向电压（电压应力）：** $U_{D} = 1.1\times 2\times \sqrt2 U_2$
          * 每个二极管承受的最大反向电压是变压器副边的峰峰值
        
        
        
    * **单相桥式整流电路**
      
        <img src="模电（十）：直流电源.assets/image-20241227232313275.png" alt="image-20241227232313275" style="zoom: 67%;" />
        
        * **输出电压有效值：** $\bar U_o = 0.9U_2\\$  
          * 为单相半波的2倍
        * **输出电流有效值：** $\bar I_{o} = \frac{\bar U_o}{R_L}=\frac{0.9U_2}{R_L}\\$ 
          * 为单相半波的2倍
        * **二极管电流应力：** $I_{D} = 1.1\frac{I_o}2\\$
          * 只有半个周期导通，但是平均电流又翻倍，所以电流应力和单相半波整流相同
        * **二极管反向电压（电压应力）：** $U_{D} = 1.1 \sqrt2 U_2$
          * 与单相半波相同



## 二、滤波电路

- 滤去交流分量，减小电压脉动，使输出平滑

- **电容滤波电路：** 利用电容的充放电作用，平滑电压
